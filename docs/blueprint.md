# **App Name**: ForgeAI

## Core Features:

- Voice Command Interpretation: Transcribe and interpret user voice commands to understand the desired tool or modification.
- Dynamic Code Generation: Generate custom-coded elements (HTML, CSS, JS) based on interpreted user voice requests, and use this tool to create logic and visuals for requested elements.
- Real-time Preview: Display a real-time preview of the AI-generated element, showcasing the code animation and the interactive block.
- Toolbox Storage: Persist generated elements in a personal toolbox, tracking storage usage and allowing deletion/modification.
- Task Management Integration: Integrate a voice-based task management system for linking tasks across workspaces.
- AI Personalization: Learn user preferences for colors, fonts, and layouts, automatically adapting outputs to match the user's style.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to reflect energy and forward-thinking.
- Background color: Dark gray (#28282B) for a coding-inspired dark theme.
- Accent color: Neon green (#39FF14) for highlights and calls to action, evocative of coding environments.
- Headline font: 'Space Grotesk', sans-serif, for a techy, scientific feel. Body font: 'Inter' sans-serif for a neutral modern look.
- Code font: 'Source Code Pro' for displaying code snippets and AI-generated code.
- Use minimalist icons with a subtle glowing effect, emphasizing the AI and coding aspects of the app.
- Incorporate subtle animations during code generation and preview updates, with a smooth fade-in effect for new blocks.
